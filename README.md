# intelligent-recruitment-assistant-demo

智能招聘助手Demo：解析JD与简历匹配度并生成报告

项目经历：我曾主导“智能招聘助手”Demo的0-1搭建。针对中小公司HR筛选简历效率低的痛点，我使用Cursor在48小时内独立开发出一个可交互的React原型。该Demo集成了大模型API，能自动解析JD与简历匹配度并生成评估报告。通过内部测试，工具将HR初步筛选耗时降低了70%，后续我将其核心逻辑抽象为可配置的“人才数据评估”模块，验证了AI在招聘场景的落地逻辑。

## 运行
```bash
pip install -r requirements.txt
python main.py
```
